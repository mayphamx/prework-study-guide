Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to main branch
  • git clone: clones repository into a new directory/copies existing git repository
  • *the git command must precede every Git statement*
  • git checkout branch-name: switches to specified branch
  • git add -A:add changed files to working directory after saving
  • git commit -m "name of file you want to commit"
  • git pull origin main: syncs and merges code from main branch to be updated with more features and content from different team members
  • git push origin "name of file":push changes to main branch (must be done after commit)
