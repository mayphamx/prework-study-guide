Prework Study Guide

HTML

  • The head element contains information about the webpage. The head element is not visible to the users.
  • The body element represents the visible content shown to the user. The body element is visible to users.

CSS

  • The margin indicates how much space we want around the outside of an element.
  • The padding indicates how much space we want around the content inside an element.
  • astrik(*) applies styling properties to all the elements visible on the page
  • commas(,) combine two selectors into a single CSS rule allowing the declarations to affect both selectors
  • img - assigns property values to any image element in the markup
  • block - assigns each image to start on a new line
  • link - link element creates the connection to the external file
  • rel - specifies the relationship between the current document and the linked document resource
  • href - specifies the location (URL) of the external resource. The href attribute is using a relative path. This means the location is derived from the current location.
  • EX.
  • card. - class selector that styles different topics that contains width, margin, border, and box-shadow properties; the period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS
  • width: 80%:; - width of the element at 80% of the available width
  • margin: 40px auto; - margin property assigns 40px to the top and bottom, and the auto value to the left and right
  • border: 5px solid gray; - border property assigns a border with a width of 5px that is solid and gray
  • box-shadow: 5px 10px #888888; - box-shadow CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas. A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color. In our example, the X offset is 5px and defines the width or extent of the shadow on the X axis (or to the right of the box); the Y offset is 10px and defines the extent on the Y axis (or just below the box)
  • section class="card" - class selector that can be assigned to all section elements in the html file that will assign the same property assignments to all sections

Git

  • *the git command must precede every Git statement*
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to main branch
  • git clone: clones repository into a new directory/copies existing git repository
  • git checkout branch-name: switches to specified branch
  • git add -A:add changed files to working directory after saving
  • git commit -m "name of file you want to commit"
  • git pull origin main: syncs and merges code from main branch to be updated with more features and content from different team members
  • git push origin "name of file":push changes to main branch (must be done after commit)
